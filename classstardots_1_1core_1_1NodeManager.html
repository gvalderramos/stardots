Detailed Description

Manages a collection of nodes, providing utilities to add, access, and query nodes.

The NodeManager class encapsulates a collection of nodes, allowing iteration, access by index, and specialized queries for nodes with the least connected inputs or outputs. This is part of the Stardots framework core.
